# Flights Data Dashboard (Streamlit)

This project is an interactive dashboard built using Streamlit to analyze flight passenger data.
It shows trends, distributions, growth patterns, and insights using different visualizations.

## Features

* Dataset preview
* KPI cards (Total records, Years covered, Total passengers)
* Passenger trend by year
* Passenger trend by month
* Histogram of passenger distribution
* Average passengers per year
* Busiest months analysis
* Box plot for passenger spread
* Cumulative passenger growth
* Yearly growth rate
* Summary statistics and correlation table
* Insights written below each chart

## Tools and Technologies

* Python
* Streamlit
* Pandas
* Matplotlib

## Project Structure


## How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```

3. The dashboard will open in your browser.
